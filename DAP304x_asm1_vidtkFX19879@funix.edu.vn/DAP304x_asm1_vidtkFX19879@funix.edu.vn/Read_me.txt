Đây là một đoạn code python để phân tích dữ liệu điểm thi của các lớp học từ các file txt. Đoạn code này thực hiện các nhiệm vụ sau:

Nhập tên file txt chứa dữ liệu điểm thi của một lớp học
Mở file txt và đọc các dòng dữ liệu
Báo cáo tổng số dòng dữ liệu hợp lệ và không hợp lệ trong file
Khai báo và xử lý đáp án cho bài thi
Đếm số lượng học sinh đạt điểm cao (>80)
Tính điểm trung bình, điểm cao nhất, điểm thấp nhất, miền giá trị và giá trị trung vị của điểm
Tìm các câu hỏi bị học sinh bỏ qua nhiều nhất và sai nhiều nhất
Viết file txt mới chứa mã số học sinh và điểm số của họ
Để sử dụng đoạn code này, bạn cần có các file txt chứa dữ liệu điểm thi của các lớp học trong cùng thư mục với file python. Bạn cũng cần cài đặt các thư viện re, pandas và numpy. Bạn có thể chạy đoạn code này bằng cách gõ lệnh python lastname_firstname_grade_the_exams.py trên dòng lệnh. Bạn sẽ được yêu cầu nhập tên file txt bạn muốn phân tích. Sau khi nhập tên file, bạn sẽ thấy kết quả xuất hiện trên màn hình và file txt mới được tạo ra.
Sau mỗi lần chạy chương trình chỉ thực hiện với 1 file lớp duy nhất, nếu bạn muốn thực thi trên 1 file lớp khác bạn cần khởi chạy lai chương trình
